Preguntas frecuentes

Lo que todo comprador pregunta

¿Mis datos salen de mi empresa?
No. Todo corre local: en tu PC (programa .exe o .bat) o en un servidor tuyo (modo web). No hay nube de terceros, no hay telemetría, nada viaja a internet. Por eso es apto para bancos, salud y datos personales.
¿Funciona con mi herramienta de BI?
Sí. Todas las tablas de gobierno se exportan como CSV, Excel, JSON o Parquet, y también se sirven por API REST. Power BI, Tableau, Looker, MicroStrategy, Qlik y Excel tienen guía paso a paso incluida en la documentación.
¿Necesito saber programar?
No. Es doble clic: el instalador .exe no requiere nada, y la versión portable .bat se instala sola la primera vez. Subís tus archivos y la plataforma perfila, mide y sugiere reglas automáticamente.
¿Y si TI no permite instalar programas?
Hay tres caminos según la política de tu empresa: instalador .exe (si permiten software), versión portable .bat que no toca el sistema (si permiten Python), o despliegue 100% web en un servidor interno. Mismas funcionalidades en los tres.
¿En qué idiomas está?
Español, inglés y portugués — en toda la superficie: el programa, esta web, la API (?lang=es|en|pt), el instalador de Windows y la documentación. El glosario guarda cada definición en los tres idiomas.
¿Puedo probarlo gratis? ¿Cuánto tarda ver resultados?
Sí: el plan Starter es gratis para siempre con datos de demo, y podés perfilar tus propios archivos desde el primer día. El índice de calidad real lo ves el día 1; el gobierno completo andando, en unos 90 días (mirá el plan de implementación acá arriba).
