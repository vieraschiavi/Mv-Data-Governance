# ⚡ Opción B · Portable .bat autoinstalable (Streamlit)

**Para empresas que NO permiten instalar .exe pero sí tienen Python.** No
instala nada en el sistema: todo vive en la carpeta del proyecto (entorno
virtual incluido). Borrar la carpeta = desinstalar.

**For companies that do NOT allow .exe installs but do have Python.** Nothing
is installed system-wide: everything lives inside the project folder
(virtual environment included). Delete the folder = uninstalled.

**Para empresas que NÃO permitem instalar .exe mas têm Python.** Nada é
instalado no sistema: tudo vive na pasta do projeto (ambiente virtual
incluído). Apagar a pasta = desinstalado.

---

## 🇪🇸 Cómo se entrega y se usa

1. Entregá a la empresa el ZIP `MVDataGovernance_OpcionB_Portable_v1.0.0.zip`
   (generado con `python packaging/build_release.py`) o la carpeta completa
   del proyecto.
2. El usuario descomprime donde quiera (Documentos, Escritorio, pendrive).
3. Doble clic en **`MV_DataGovernance.bat`**:
   - **Primera vez**: crea solo el entorno `.venv` dentro de la carpeta e
     instala las dependencias (2–5 min, requiere internet una única vez).
   - **Siempre**: abre el programa en el navegador. La ventana negra queda
     abierta mientras el programa corre; cerrarla apaga el programa.
4. Para la API de BI: doble clic en **`MV_DataGovernance_API.bat`**.

Requisito: [Python 3.10+](https://www.python.org/downloads/) con la casilla
**"Add Python to PATH"** marcada al instalarlo.

## 🇬🇧 How it is delivered and used

1. Give the company the `MVDataGovernance_OpcionB_Portable_v1.0.0.zip`
   (built with `python packaging/build_release.py`) or the full project
   folder.
2. The user unzips it anywhere (Documents, Desktop, USB drive).
3. Double-click **`MV_DataGovernance.bat`**:
   - **First run**: creates the `.venv` environment inside the folder and
     installs dependencies (2–5 min, needs internet once).
   - **Always**: opens the program in the browser. The black window stays
     open while the program runs; closing it stops the program.
4. For the BI API: double-click **`MV_DataGovernance_API.bat`**.

Requirement: [Python 3.10+](https://www.python.org/downloads/) with
**"Add Python to PATH"** ticked during install.

## 🇧🇷 Como é entregue e usado

1. Entregue à empresa o ZIP `MVDataGovernance_OpcionB_Portable_v1.0.0.zip`
   (gerado com `python packaging/build_release.py`) ou a pasta completa do
   projeto.
2. O usuário descompacta onde quiser (Documentos, Área de trabalho, pendrive).
3. Duplo clique em **`MV_DataGovernance.bat`**:
   - **Primeira vez**: cria o ambiente `.venv` dentro da pasta e instala as
     dependências (2–5 min, precisa de internet uma única vez).
   - **Sempre**: abre o programa no navegador. A janela preta fica aberta
     enquanto o programa roda; fechá-la encerra o programa.
4. Para a API de BI: duplo clique em **`MV_DataGovernance_API.bat`**.

Requisito: [Python 3.10+](https://www.python.org/downloads/) com a caixa
**"Add Python to PATH"** marcada na instalação.
